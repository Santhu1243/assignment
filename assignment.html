Ordered List of HTML Tags

  1. html - The root element of an HTML document.
  2. head - Contains meta information, links to stylesheets, and the document title.
  3. title - Sets the title of the web page displayed in the browser tab.
  4. body - Contains all the visible content of a web page.
  5. h1 to h6 - Defines headings
  6. p - Defines a paragraph.
  7. a - Creates hyperlinks to navigate to other pages or websites.
  8. img - Embeds an image in the webpage.
  9. ul - Creates an unordered (bulleted) list.
  10. ol - Creates an ordered (numbered) list.
  11. li - Defines a list item inside ul, ol, or menu
  12. table - Defines a table.
  13. tr - Defines a table row.
  14. td - Defines a table data cell.
  15. th - Defines a table header cell.
  16. form - Creates a form for user input.
  17. input - Defines an input field inside a form.
  18. button - Adds a clickable button.
  19. div - A container used for structuring content.
  20. span - An inline container for styling specific parts of text.
  21. nav - Represents a navigation section.
  22. header - Defines a header section for a webpage or section.
  23. footer - Defines a footer section with additional information.
  24. section - Represents a distinct section of a document.
  25. article - Represents an independent piece of content.
  26. aside - Represents additional content like sidebars.
  27. audio - Embeds audio content.
  28. video - Embeds video content.
  29. canvas - Provides an area for drawing graphics using JavaScript.
